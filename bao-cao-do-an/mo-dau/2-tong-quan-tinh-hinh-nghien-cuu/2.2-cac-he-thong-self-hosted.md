# 2.2. Các Hệ Thống Media Server Tự Quản Lý (Self-Hosted)

Song song với các nền tảng thương mại, một cộng đồng phát triển phần mềm mã nguồn mở đã tạo ra các hệ thống media server tự quản lý cho phép người dùng xây dựng nền tảng streaming riêng trên phần cứng cá nhân hoặc máy chủ thuê. Ba đại diện nổi bật nhất trong phân khúc này là **Plex Media Server**, **Jellyfin**, và **Emby**.

Plex Media Server là giải pháp self-hosted phổ biến nhất, cung cấp giao diện người dùng hoàn thiện, hỗ trợ đa nền tảng (web, mobile, smart TV), tích hợp metadata tự động từ TMDB và TVDB, và tính năng transcoding on-the-fly. Tuy nhiên, Plex theo mô hình freemium — nhiều tính năng quan trọng như sync offline và remote access yêu cầu đăng ký Plex Pass trả phí. Jellyfin là fork mã nguồn mở hoàn toàn miễn phí của Emby, ra đời năm 2018 sau khi Emby chuyển sang mô hình thương mại. Jellyfin cung cấp hầu hết các tính năng tương đương nhưng không có lớp backend hỗ trợ encode đa codec chuyên biệt và thiếu tính linh hoạt trong việc tùy biến pipeline xử lý video.

Điểm chung của các hệ thống trên là chúng được thiết kế để phục vụ đa mục đích, dẫn đến kiến trúc monolithic và khó tùy biến sâu ở tầng encode. Đặc biệt, khả năng tích hợp hàng đợi encode phân tán (như BullMQ), hỗ trợ nhiều cloud storage provider (S3, Azure Blob, Google Drive), và xây dựng pipeline ABR đầy đủ từ source file đến MPEG-DASH segment là những khoảng trống mà các giải pháp hiện tại chưa cung cấp dưới dạng một framework có thể mở rộng tự do.
