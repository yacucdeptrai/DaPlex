# 3.5.2. StreamManifest — Định Dạng Trung Gian

`StreamManifest` là JSON object trung gian do DaPlex-API trả về thay cho MPD XML trực tiếp. Định dạng này chứa metadata về tất cả video track và audio track của một bản encode, phục vụ cả DASH lẫn HLS fallback từ cùng một bộ file MP4. `VideoPlayerService` phía frontend đọc `StreamManifest` và chuyển đổi thành `ParsedDashManifest` trước khi truyền vào dash.js.

---

## Cấu Trúc Interface

Các interface chính của `StreamManifest`:

```typescript
interface StreamManifest {
  hlsVersion: number;
  videoTracks: HlsVideoTrack[];   // mỗi resolution là một track
  audioTracks: HlsAudioTrack[];   // mỗi codec/ngôn ngữ là một track
  totalDuration: number;
  targetDuration: number;
  mediaSequence: number;
  playlistType: string;
}

interface HlsVideoTrack {
  codecID: number;       // VideoCodec enum: H264=1, H265=2, VP9=4, AV1=8
  width: number;
  height: number;
  bandwidth: number;
  frameRate: number;
  codec: string;         // chuỗi codec cho MPD, vd: "avc1.640028"
  mimeType: string;
  dashSegment: DashSegmentGroup;  // initRange + indexRange
  hlsSegment: HlsSegmentGroup;   // segments + byterange cho HLS fallback
  uri: string;           // đường dẫn file MP4 trên storage
}
```

> Thiết kế tách biệt `dashSegment` và `hlsSegment` cho phép cùng một manifest phục vụ cả hai giao thức từ cùng một file MP4 gốc — không cần tạo file riêng cho HLS hay DASH.
